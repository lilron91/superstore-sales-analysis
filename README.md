# Superstore Sales Analysis

## Overview
SQL-based analysis of the Kaggle Superstore dataset using MS SQL Server.
Includes data modeling, normalization and business-focused queries.

## Database Structure
- customers
- products
- locations
- orders
- order_details

## Analyses
1. Monthly Sales Trend - Tracks revenue performance over time to identify seasonal patterns and growth trends
2. Top 10 Customers by Revenue - Identifies highest revenue generating customers to support retention and upsell strategies
3. Sales by Region Year over Year - Compares regional performance year over year to identify growth and decline trends
4. Sub-Category Revenue Breakdown - Reveals which product sub-categories drive the most revenue to support assortment decisions
5. RFM Customer Segmentation - Helps identify high-value customers and those at risk of churning
6. Shipping Performance by Ship Mode - Identifies average delivery time per shipping method to optimize logistics and customer satisfaction

## Tools Used
- MS SQL Server
- SQL Server Management Studio
